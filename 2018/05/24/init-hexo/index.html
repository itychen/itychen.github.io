hexo+github搭建个人博客

发表于 2018-05-24 | 分类于 hexo+github搭建个人博客

简介

Hexo(中文官方网站)是一个快速, 简洁且高效的博客框架.让上百个页面在几秒内瞬间完成渲染.

环境配置

Hexo是基于node.js的, 所以我们在安装它之前需要用到npm安装工具, 这个工具是 node.js 安装包的工具, 所以, 我们先要安装 node.js.

安装Node.js

用来生成静态页面。移步Node.js官网下载, 一路安装即可。

安装Git

用来将本地Hexo内容提交到Github上。

建站

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。

$ hexo init 
$ cd 
$ npm install
新建完成后，指定文件夹的目录如下： 
.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes
在指定文件夹下执行命令： 
$ npm install

执行如下命令，开启hexo服务器：


$ hexo clean
$ hexo g
$ hexo s
